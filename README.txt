Техническое задание
Вам нужно написать код для трёх функций.
    Функция insertClient() — добавляет запись в таблицу clients. Возвращает идентификатор добавленной записи и ошибку. На вход принимает:
        db — указатель на объект типа sql.DB;
        client — объект типа Client с данными о клиенте.

    Функция updateClientLogin() — обновляет поле login у записи с заданным id в таблице clients. Возвращает ошибку. На вход принимает:
        db — указатель на объект типа sql.DB;
        login — логин клиента;
        id — идентификатор записи.

    Функция deleteClient() — удаляет запись из таблицы clients по заданному id. Возвращает ошибку. На вход принимает:
        db — указатель на объект типа sql.DB;
        id — идентификатор записи.


В функции main() создаётся объект типа Client и сохраняется в переменной newClient. 
Для него нужно заполнить следующие поля:
    FIO — ФИО клиента;
    Login — логин клиента;
    Birthday — день рождения клиента в виде строки в формате YYYYMMDD (например, 19910225);
    Email — почта клиента.

Для переменной newLogin нужно задать новый логин клиента.


В функции main происходит следующее:
    - происходит подключение к БД;
    - вызывается функция insertClient() для добавления новой записи;
    - вызывается функция selectClient(), которая выводит на консоль данные клиента по его идентификатору;
    - вызывается функция updateClientLogin() для обновления логина;
    - снова вызывается функция selectClient(), чтобы вывести обновлённые данные клиента;
    - вызывается функция deleteClient(), которая удаляет клиента;
    - вызывается функция selectClient(), которая должна вернуть ошибку, поскольку записи уже нет.


В итоге на консоли должно вывестись 3 строки:
    - первая с исходными данными клиента;
    - вторая с изменённым логином;
    - третья строка с ошибкой sql: no rows in result set, которая говорит о том, что клиента в таблице уже нет.
